# 구현 계획: 사용자 맞춤형 대시보드

## 개요

하드코딩된 모든 데이터를 localStorage 기반 사용자 관리 데이터로 전환한다. useLocalStorage 커스텀 훅을 중심으로 App.jsx에서 중앙 상태 관리를 구현하고, 모달/토스트/빈 상태 등 공통 UI 컴포넌트를 추가한 뒤, 각 페이지를 props 기반으로 리팩터링한다.

## 참고

- `*` 표시된 태스크는 선택 사항이며 빠른 MVP를 위해 건너뛸 수 있다
- 각 태스크는 추적 가능성을 위해 구체적인 요구사항을 참조한다
- 체크포인트에서 점진적 검증을 수행한다
- 모든 새 컴포넌트는 기존 CSS 커스텀 속성을 활용하여 일관된 디자인을 유지한다
